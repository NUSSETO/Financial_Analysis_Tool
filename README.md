# Financial Analysis & Optimization Tool

## Overview
An end-to-end data science application built with Python and Streamlit that assists investors in decision-making through statistical modeling.

## Key Features
* **Stock Price Forecasting:** Utilizes **Monte Carlo Simulations** (Geometric Brownian Motion) to model future price uncertainty and quantify risk (VaR, CVaR).
* **Portfolio Optimization:** Applies **Modern Portfolio Theory (MPT)** and **convex optimization** to construct the Efficient Frontier and maximize the Sharpe Ratio.
* **Interactive Dashboard:** Real-time data visualization using Plotly.

## Tech Stack
* **Core:** Python, NumPy, Pandas, SciPy
* **Visualization:** Plotly, Streamlit
* **Data Source:** Yahoo Finance API (yfinance)
